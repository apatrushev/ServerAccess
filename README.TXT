1.INSTALLATION INSTRUCTIONS

1.1. Windows installation
1.1.1. Install JDK.
1.1.2. Use installer serveraccess.exe.

1.2. LINUX
2.1.1. Install JDK.
2.1.2. Unpack  serveraccess.tgz.
2.1.3. Install putty.
2.1.4. Install gftp.

2. PURPOSE
Serveraccess is intended to provide easy and seamless access
to multiple servers. It uses various access protocols,
such as ssh2, ftp, and rdp. It uses accounts stored in either
file system or on a web server. It can make local port forwarding
on demand or by predefined settings. It can make connections
through several tunnels.
